# Source Layout

- `components/`
  Shared presentational UI.
- `hooks/`
  Shared React hooks.
- `navigation/`
  Route names and navigator composition.
- `redux/`
  Global app state and provider logic.
- `screens/`
  Top-level screen modules.
- `services/`
  Domain logic and Supabase integration.
- `types/`
  Shared type placeholders and future typing expansion.
- `utils/`
  Theme tokens and formatting helpers.
